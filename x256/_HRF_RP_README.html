Get started

How do I install High Rossferry Resource Pack?

It's very easy.

  1. Download MC Patcher or OptiFine and follow the instructions.
  2. Download High Rossferry Resource Pack.
  3. Go to .minecraft folder.
  4. Copy HRF_RP_1.6.2_x64 or x256.zip into "resourcepacks" folder.
  5. Launch the game.
  6. Go to Options... > Resource Packs and select High Rossferry Resource Pack.

How do I update High Rossferry Resource Pack?

Upgrading is usually the process of re-installing the new version over the old version.
